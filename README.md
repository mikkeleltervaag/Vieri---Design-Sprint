# Vieri Apps 2026 Design Suggestion

## Viewing the demo

- **demo.html** - Open directly in a browser (double-click). Everything is self-contained, no server needed.
- **index.html** - Requires a local server (e.g. `npx serve` or VS Code Live Server) since it uses `fetch()` to load components.

Both versions require the `images/` and `svg/` folders to be in the same directory.
